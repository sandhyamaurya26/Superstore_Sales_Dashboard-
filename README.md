# Superstore Sales Dashboard | Microsoft Excel

> An interactive Excel dashboard built to analyze Superstore sales performance using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

---

## Project Overview

This project presents an interactive sales dashboard developed in **Microsoft Excel** to analyze business performance across multiple dimensions, including sales, profit, quantity, customer segments, shipping methods, and geographical regions.

The dashboard enables users to explore the data dynamically through interactive slicers, making it easier to identify trends, compare performance, and support business decision-making.

---

## Business Insights

- Technology contributes the highest overall profit.
- Consumer segment accounts for the largest share of sales.
- West region generates the highest sales.
- Standard Class is the most frequently used shipping mode.
- Higher discounts often reduce overall profitability.
- Sales are concentrated in a few high-performing states.
